## Overview
This program simulates a simplified version of the card game Blackjack, allowing a user to play against a dealer.

---

## Code Overview

### Purpose
The objective of this Java program is to model and simulate a game of Blackjack using object-oriented design and basic data structures.

---

### Key Components

- `Card`  
  - Represents an individual playing card with a suit and value  

- `Deck`  
  - Stores and manages a collection of cards  
  - Handles shuffling and dealing  

- `Player`  
  - Manages the user’s hand and actions  

- `Dealer`  
  - Implements dealer behavior and rules  

- `Main`  
  - Controls the game flow and user interaction  

---

### Key Methods

- Game Initialization  
  - Creates and shuffles the deck  
  - Deals initial cards to player and dealer  

- Player Turn  
  - Allows the player to "hit" or "stand"  
  - Updates hand values accordingly  

- Dealer Turn  
  - Dealer draws cards based on predefined rules  

- Game Outcome  
  - Compares hands and determines winner  

---

## Data Structures Used
- Stack (implemented using arrays)  
  - Used to manage the deck of cards as they are drawn and reshuffled  

---

## Pre-requisites
- Java development environment (IDE or terminal)
- Basic understanding of Java classes and execution
